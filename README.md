Project Title: Tata Group Data Visualisation Analysis
Project Overview: Completed a virtual internship with Tata Group via Forage. The goal was to provide business insights to the CEO and CMO regarding global sales trends and product demand.
Key Tasks:
Data Cleaning: Used Excel to process 500,000+ rows, removing invalid entries (negative quantities/prices).
Time-Series Analysis: Visualised 2011 revenue trends to identify seasonal peaks.
Global Mapping: Created a geographic dashboard to track product demand outside the UK.
Customer Insights: Identified top 10 high-value customers for targeted marketing.
Tools Used: * Excel (Data Cleaning & Analysis)
